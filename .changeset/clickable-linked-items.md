Make the linked artifacts in the diagram-node and message drill-downs clickable.

Tapping a linked decision / action / question / point inside a node drill or a
message drill now stacks that artifact's full view (text, sub, and evidence) on
top of the current modal, so the evidence cross-link is navigable end-to-end
instead of just being displayed. The rows adopt the themed hover/focus
interaction recipe.
